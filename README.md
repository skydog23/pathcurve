Development
===========

Prerequisites
-------------

* install nodejs and npm per instructions at http://nodejs.org
* `sudo npm install -g webpack`
* `sudo npm install -g webpack-dev-server`


Running
-------

* `npm run dev`
* browse to http://localhost:8080
